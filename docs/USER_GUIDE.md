## Troubleshooting

### "Failed to fetch video info"

**Possible causes:**
- ‚ùå Invalid URL
- ‚ùå Private or restricted video
- ‚ùå Video has been deleted
- ‚ùå Platform temporarily unavailable

**Solutions:**
1. Check the URL is correct and complete
2. Make sure the video is public
3. Try a different video
4. Wait a few minutes and try again

---

### Download doesn't start

**Solutions:**
1. Check your internet connection
2. Try a different quality option
3. Disable popup blockers
4. Try a different browser

---

### Video plays instead of downloading

**On Desktop:**
- Right-click the download button
- Select "Save link as..."

**On Mobile:**
- Long-press the download button
- Select "Download link"

---

### Download is very slow

**Reasons:**
- Your internet speed
- Server load
- Video file size

**Tips:**
- Choose lower quality for faster downloads
- Wait for better network conditions
- Try again during off-peak hours

### Can I download private videos?

No, only public videos are accessible.

Some platforms block automated downloads or have geo-restrictions.

Currently, only single videos are supported. Download each video individually.

No official limit, but please use responsibly to avoid server overload.

Not currently supported. This feature may be added in the future.

### Does it work on all browsers?

Yes! Supports:
- Chrome, Firefox, Safari, Edge
- Mobile browsers

### Can I use this offline?

No, an internet connection is required to fetch and download videos.
